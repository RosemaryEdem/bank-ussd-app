Bank USSD Application
## About
This is a basic Bank USSD application built with Python. It simulates how banking USSD services work on mobile phones.

## Features
1. PIN verification
2. Check account balance
3. Buy airtime
4. Transfer money
5. Change PIN
6. Exit application
   
## Concepts Used
1. Python programming language
2. Variables
3. if/elif/else statements
4. while loops
5. User input handling

## How to Run
On Linux/Mac:

```Bash
python3 bank_ussd.py
```

Or on Windows:

```Bash
py bank_ussd.py
```
